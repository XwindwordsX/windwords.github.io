数学（文科）

集合

集合中元素的性质

我们把研究的对象统称为元素，把一些元素组成的总体叫做集合

  • 我们要求集合的定义是明确的（确定性）。即对于每个被考察的对象，它要么是这个集合的元素，要么不是这个集合的元素。比如“所有体重超过200斤的人”可以构成一个集合，而“所有胖子”则不能，因为哪些人是“胖子”，哪些人不是“胖子”并没有一个明确的标准。又比如“身高在185cm以上的人”可以构成一个集合，而“所有高个子”则不能。

  • 我们要求集合中的元素必须是互异的（互异性）。即集合中的元素不能重复出现，有重复出现元素的必定不是集合。

  • 构成集合的元素可以是无序的（无序性）。即集合内的元素可以任意以任意顺序排列。

集合的相互关系及其表示法

集合的相互关系

若组成集合的元素完全相同，那么，我们就说，这两个集合是相等的。若集合$ A $与集合$ B $相等，记作：$ A=B $。

若元素$ a $是集合$ A $中的一个元素，我们就说$ a $属于$ A $，记作：$ a\in A $。否则，则说$ a $不属于$ A $，记作：$ a\notin A $。

若集合$ A $的所有元素都在集合B中，我们就说“$ A $含于$ B $”或“$ B $包含于$ A $”，记作：$ A\subseteq B $。若$ A\subseteq B $且$ A\neq B $，则称$ A $是$ B $的真子集，亦称$ A $真包含于$ B $，或$ B $真包含$ A $，记作：$ A\subset B $。

集合的表示法

集合通常用大写的拉丁字母$ A, B, C $等表示，而集合的元素通常用小写的拉丁字母$ a, b, c $等表示。

  • 描述法：$ \left\{2410班全体成员\right\} $

    • 例子 :

      问题：方程$ x^2-3x+2=0 $的解的集合

      答案：$A=\left\{x\in R|x^2-3x+2=0\right\} $

  • 列举法：$ \left\{1, 2, ……, 10\right\} $

    • 例子：

      问题：方程$ x^2-3x+2=0 $的解的集合

      答案：$ \left\{1, 2\right\} $

  • 自然语言法

中学阶段常用数集

  • 自然数集（非负整数集）：$ N $
  • 正整数集：$ N^* $ 或 $ N_+ $
  • 整数集：$ Z $
  • 有理数集：$ Q $
  • 实数集：$ R $

空集

不含有任何元素的集合称为空集，记作$ \emptyset $或$ \left\{\right\} $。

空集是任何集合（包括空集）的子集。

Na（钠）

  • 钠与水反应：$ \ce{2Na + H2O->2NaOH + H2} $

  • 钠与空气反应：

    • 常温：$ \ce{4Na + O2->2Na2O} $
    • 加热：$ \ce{2Na + O2->[\Delta]Na2O2} $

  • 钠与盐溶液反应：

    $ \ce{K, Ca, Na} $活泼金属与盐溶液反应，不能置换出盐中金属。

    • 使用$ \ce{CuSO4} $：$ \ce{2Na + 2H2O + CuSO4 -> Cu(OH)2 v + Na2SO4 + H2 ^} $
    • 使用$ \ce{NH4Cl} $：$ \ce{2Na + 2NH4Cl -> 2NaCl + 2NH3 + H2} $

  • 钠在空气中变质过程：

    $ \ce{4Na + O2 -> 2Na2O} $

    $ \ce{Na2O + H2O -> 2NaOH} $

    $ \ce{2NaOH + CO2 + 9H2O -> Na2CO3.10H2O} $

    $ \ce{Na2CO3.10H2O -> Na2CO3 + H2O} $（风化）

  • 氧化钠与水反应：$ \ce{Na2O + H2O -> 2NaOH} $

  • 氧化钠与二氧化碳反应：$ \ce{Na2O + CO2 -> Na2CO3} $

  • 氧化钠与盐酸反应：$ \ce{Na2O + 2HCl -> 2NaCl + H2O} $

  • 氧化钠在空气中反应：$ \ce{2Na2O + O2 -> 2Na2O2 } $（$ \ce{Na2O} $不稳定，易被氧化）

  • 过氧化钠与盐酸反应：$ \ce{2Na2O2 + 4HCl -> 4NaCl +2H2O +O2 ^} $

  • 过氧化钠与二氧化碳反应：$ \ce{2Na2O2 + 2CO2 -> 2Na2CO3 + O2} $

  • 过氧化钠与水反应：$ \ce{2Na2O2 + 2H2O -> 4NaOH +O2 ^} $

    实质：

    $ \ce{Na2O2 + H2O -> H2O2 + 2NaOH} $（$ \ce{Na2O2} $具有漂白性）

    $ \ce{ 2H2O2 -> 2H2O + O2 ^} $

  • 过氧化钠与氯化铁反应：$ \ce{6Na2O2 + 6H2O + 4FeCl3 -> 4Fe(OH)3 v + 12NaCl +3O2 ^} $

    实质：

    $ \ce{Na2O2 + 2H2O -> 4NaOH + O2 ^} $

    $ \ce{3NaOH + FeCl3 -> Fe(OH)3 v + 3NaCl} $
